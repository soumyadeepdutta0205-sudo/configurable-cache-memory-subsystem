- Configurable cache memory subsystem
- Direct-Mapped and 2-Way Set Associative Cache
- Write-Back policy with Dirty Bit handling
- LRU Replacement Policy
- Performance Counters
- Complete RTL-to-GDSII implementation

- Floorplanning
- Placement and CTS

- Timing Closure
